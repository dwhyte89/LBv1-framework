# Persona Layer (Conceptual)

LBV1 separates cognition into two layers:

## Core Brain (Immutable)
- Execution logic
- Safety rules
- Permission enforcement
- Memory handling
- Model routing

## Persona Layer (Swappable)
- Tone
- Directness
- Verbosity
- Structure preference
- Decision framing style

The persona layer does not modify core execution rules.

No silent personality drift.
User preferences are explicit and adjustable.
